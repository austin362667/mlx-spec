# mlx-spec
Speculative Decoding in MLX


## Setup

1. Install [`uv`](https://docs.astral.sh/uv/)
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. Create `venv`
```bash
uv venv --python 3.11
```

3. Install dependencies
```bash
uv sync
```

## Quick start

```bash
uv run python -m benchmark
```
